# Sales & Delivery Performance Analysis

## 📌 Project Overview
This project focuses on analyzing sales, profit, and delivery performance to identify business insights and operational improvement areas using Python and Power BI.

The analysis helps understand:
- Regional sales trends
- Profitability by category
- Delivery performance issues
- Impact of delays on profit

---

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Power BI
- Excel
- Jupyter Notebook

---

## 📊 Dashboard Features
### Executive Overview
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Sales by Region

### Regional Analysis
- Sales by Region
- Profit by Region
- Category-wise Sales Analysis

### Delivery Performance
- Delayed Orders Analysis
- Delay Distribution
- Region-wise Delivery Insights

---

## 🔍 Key Business Insights
- South region recorded highest sales and profit.
- Furniture category achieved highest profit margin.
- West region experienced maximum delivery delays.
- Delivery delays negatively impacted profitability.
